# Система управления рецептами - консольное приложение
**Домашнее задание №2. Технологии програмирования, БИ**

**Краткое описание проекта:**
Проект представляет собой объектно-ориентированную систему на языке Python для работы с кулинарными рецептами. Программа позволяет создавать отдельные ингредиенты, объединять их в рецепты (в том числе диетические), масштабировать граммовки в зависимости от количества порций, а также автоматически формировать список покупок без дубликатов. 

## Использование

**Требования:**
* Установленный интерпретатор Python 3
* Библиотека для тестирования (устанавливается командой `pip install pytest`)

**Как запускать код:**
Проект реализован в виде модульной структуры в файле `recipes.py`. Для использования функционала создайте исполняемый файл (например, `main.py`) в той же директории и импортируйте необходимые классы:

```python
from recipes import Ingredient, Recipe, ShoppingList, DietaryRecipe

# Пример создания рецепта:
my_recipe = Recipe("Пицца")
my_recipe.add_ingredient(Ingredient("Мука", 500.0, "гр"))
```

**Как запускать тесты:**
Логика классов полностью покрыта автоматическими unit-тестами в файле `test_recipes.py`. 
Для запуска проверки откройте терминал в корневой папке проекта и выполните классическую команду:
```bash
python -m pytest
```
Если вы хотите увидеть подробный отчет о прохождении каждого отдельного теста, используйте команду с флагом `-v`:
```bash
python -m pytest -v
```

## Установка

Для того чтобы развернуть проект локально, выполните следующие команды в терминале:

```bash

```
2. Перейдите в папку с проектом:
```bash

```
3. Установите необходимые зависимости:

4. Запустите тесты командой:
